IMPOSTOR HINT — SPLIT FILE VERSION

Files:
- index.html: page structure
- style.css: graphics and layout
- app.js: game logic and features
- manifest.webmanifest: installable app settings
- service-worker.js: offline caching
- icons: home-screen graphics

This version includes:
- Random All Categories
- Built-in categories
- Custom saved categories
- Impostor category hints
- Offline support

To update GitHub:
1. Upload index.html, style.css, app.js, and service-worker.js.
2. Replace the existing files when prompted.
3. Commit with a message such as: Add split app files and random category
